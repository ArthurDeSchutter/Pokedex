# Pokedex :zap:

this project is a little showcase project, nothing otherworldly just a quick project

## Installation :computer:
clone the project and run

```cmd
npm install
node server.js
```
to edit the react project
```cmd
npm install
node server.js
cd pokedex
npm install
npm start
```

to build the react project
```cmd
cd pokedex
npm install
npm run build
```
